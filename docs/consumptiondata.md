The api returns consumption-data as json-object (see below).

`date: '2016-06-01T00:00:00+0000'` means this are the consumption data from 01.06.2016 - 31.06.2016  

The average value is calculated like:
`average = (tC / tS) * uS`

tC = propertyTotalConsumption (per consumption type)  
tS = propertyTotalSize  
uS = unitSize
